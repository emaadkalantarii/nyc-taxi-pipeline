# 🚕 NYC Taxi Analytics Pipeline

> An end-to-end batch data engineering pipeline built with PySpark, Apache Airflow, PostgreSQL, dbt, Docker, and GitHub Actions CI/CD.

## 🛠️ Tech Stack

| Layer | Tool |
|---|---|
| Ingestion & Processing | PySpark |
| Orchestration | Apache Airflow |
| Storage | PostgreSQL |
| Transformation | dbt |
| Data Quality | Great Expectations |
| Containerization | Docker + Docker Compose |
| CI/CD | GitHub Actions |
| Dashboard | Streamlit |

## 📦 Dataset

NYC Yellow Taxi Trip Records — sourced from the NYC Taxi & Limousine Commission (TLC).
